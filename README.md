# ChatGPT Clone UI (Static)

A clean, minimalist, and responsive chatbot interface built using HTML and CSS.

## ✨ Features
- **Sidebar:** Navigation for chat history (UI only).
- **Dark Mode:** Styled with ChatGPT's signature dark theme.
- **Responsive:** Optimized for Chromebooks, tablets, and desktops.
- **Flexbox Layout:** Ensures the input bar stays pinned to the bottom.

## 🛠️ Built With
- HTML5 (Structure)
- CSS3 (Flexbox & Styling)
- GitHub Pages (Deployment)
